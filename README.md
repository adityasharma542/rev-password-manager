# rev-password-manager
A secure Java console-based Password Manager using JDBC, Oracle Database, Log4j, and JUnit with OTP-based account recovery.
A secure Java console-based Password Manager application that allows users
to store, manage, and recover passwords using JDBC, Oracle Database,
Log4j logging, JUnit testing, and OTP-based verification.
## Technologies Used
- Java
- JDBC
- Oracle Database
- Log4j
- JUnit
- IntelliJ IDEA
## Features
- User Registration & Login
- Master Password Encryption
- Add / View / Delete Passwords
- Search passwords by account name
- Security Question for account recovery
- OTP-based password reset
- Logging using Log4j
- Unit testing with JUnit
## Database Schema
- users
- passwords
- security_questions
